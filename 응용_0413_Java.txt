자바프로젝트 이름과 깃허브 레포지토리 이름은 같아야 된다

변수 : 데이터를 저장하는 공간
자료형 : 데이터가 어떤 값인지에 따라 사용하는 값
 - byte : 0과1을 표현 : 1byte
 - short :
 - int : 정수를 표현 : 4byte
 - long : 
 - float / double : 실수를 표현 : 4 / 8 byte 
 - char : 문자 표현 : 2byte
 - String : 
 - boolean : true 또는 false를 표현 : 1bit
변수생성
 - 자료형 변수명; 또는 자료형 변수명 = 값;
 - 1number(x), n22u11m33(o)
 - changenumber => change_Number 해두면 이해하기 좋다
자료형 변환
  - 특정 자료형을 다른 잘료형으로 변환하는 것
상수(final)
 - 한 번 저장하면 변경할 수 없다
 - 상수의 이름은 대부분 전체 대문자 표기
